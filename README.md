# music-engineering-portfolio
Student and researcher projects

## Add a project from a submission issue

1. Open the repository's **Actions** tab.
2. Select **Create Project From Issue**.
3. Click **Run workflow**, enter the submission issue number, and run it.
4. Open the compare link that the workflow posts as a comment on the issue.
5. Review the generated project file and downloaded images, create the pull request, and merge it.

The workflow downloads the issue attachments into `assets/images/projects/`, assigns the first upload as the thumbnail, creates the project Markdown file, and places additional images according to simple notes such as `diagram.png should appear after the first paragraph.`
